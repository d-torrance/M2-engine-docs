104// to create the frame:
105
106// 1. Fill in level 0 - (Take the degree information from the target mInputModGB)
107// 2. Fill in level 1 - (This is really just the info on the entries of mInputModGB)
108// This is more difficult the GB is not known out to that point yet.
109// 3. Fill in level 2 - This requires us to find all the spairs on module elements vs
110// elements of the ring Groebner basis
111// 4. Rinse and repeat.
112
113// detailed notes on a basic example
114// consider the right ideal (yyx,yxx)R with R = QQ<|x,y|>/(x*y - y*x)
115
116// Step 1:
117// level 0: right module generated by e_1
118
119// Step 2:
120// level 1: f_1 --> e_1yyx, f_2 --> e_1yxx
121
122// other gb elements? (e_1yyx) * y - (e_1 yy)*(x*y - y*x) = e_1 yyyx
123// (e_1yxx) * y - (e_1 yx)*(x*y - y*x) = e_1 yxyx ~> e_1yyxx - (e_1yyx)*x = 0
124
125// f_3 --> e_1 yyyx, etc (another f_j for each e_i y^j x for j >= 3)
126
127// Level 2:
128
129// g_1 --> f_1.y = e_1.yyx.y <~~> e_1.yyxy = e_1.yyyx, now have to add f_3 to level 1
130
131// g_1 --> f_1.y = e_1.yyx.y <~~> f_1.y - f_3 Claim: this element maps to zero under the previous map
132// g_1 will have label: e_1.yyx.y e_1yyxy - e_1yyyx = e_1yy.(xy - yx) = 0
133
134// g_2 --> f_2.y = e_1.yxx.y <~~> f_2.y - f_1.x e_1yxxy - e1yyxx = 0
135// g_2 will have label: e_1.yxx.y
136
137// g_3 --> f_3.y = e_1.yyyx.y <~~> f_3.y - f_4 (etc)
138// g_3 will have label: e_1.yyyx.y
139
140// for a term, we have: label - product of all leadTerms all the way back to level 0, e.g. e_1.yyyx.y (without the dots)
141
142// to compare
143// g_1.m_1; g_1 = f_1.v_1
144// g_2.m_2; g_2 = f_2.v_2, f_1 and f_2 are further back etc
145
146// first compare total monomials of g_1.m_1 and g_2.m_2. if there is a winner, take that term.
147// if there is a tie at this stage,
148
149// example:
150// level 0: e_1
151// level 1: f_1 = e_1.xy, f_2 = e_1.y^3, f_3 = e_1.yx
152// level 2: g_1 = f_1.xyy g_2 = f_1.xxy, g_3 = f_2.x
153
154// How to compare f_i.m and f_j.n: f_1.xxx vs f_2.xy
