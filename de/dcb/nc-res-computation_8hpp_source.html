111 // 4. Rinse and repeat.

113 // detailed notes on a basic example
114 // consider the right ideal (yyx,yxx)R with R = QQ<|x,y|>/(x*y - y*x)
115 
116 // Step 1:
117 // level 0: right module generated by e_1

119 // Step 2:
120 // level 1: f_1 --> e_1yyx, f_2 --> e_1yxx

122 // other gb elements? (e_1yyx) * y - (e_1 yy)*(x*y - y*x) = e_1 yyyx
123 // (e_1yxx) * y - (e_1 yx)*(x*y - y*x) = e_1 yxyx ~> e_1yyxx - (e_1yyx)*x = 0
124 
125 // f_3 --> e_1 yyyx, etc (another f_j for each e_i y^j x for j >= 3)
126 
127 // Level 2:
128

132 // g_1 will have label: e_1.yyx.y e_1yyxy - e_1yyyx = e_1yy.(xy - yx) = 0
133
