267  // A = m x n 'this' is factored as A = LQUP, where
268  // L is m x m lower unit triangular
269  // U is m x n upper triangular
270  // Q is a m x m permutation matrix
271  // P is an n x n permutation matrix
272  // However, the result is constructed in an abbreviated, encoded format:
273  // First, the entries of A are modified:
274  // L and U are placed into A
275  // Two integer arrays are returned:
276  // The first represents Q, giving the rows of U ??
277  // The second represents P, which tells which columns of A have non-zero
278  // pivots
279  // (and, the elements below the main diagonal in these columns form the
280  // lower part of L).
