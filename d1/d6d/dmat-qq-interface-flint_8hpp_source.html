6 // This class is designed to use DMat<M2::ARingQQ>, which stores elements as gmp
7 // ints
8 // This sets up flint fmpq_mat matrices, and provides translation. This is
9 // significantly faster than doing the operations in a naive manner.
10 // This will become un-needed once DMat<ARingQQ> starts using flint
11 // integers/rationals.
