9 // Class: Subsets
10 // Manipulate p-subsets of 0..n-1, for fixed n, but possibly several p.
11 //
12 // Encoding and decoding is done as follows:
13 // Consider e.g. 3-subsets:
14 // These are
15 // 0 = 0 1 2
16 // 1 = 0 1 3
17 // 2 = 0 2 3
18 // 3 = 1 2 3
19 // 4 = 0 1 4
20 // 5 = 0 2 4
21 // 6 = 0 3 4
22 // 7 = 1 2 4
23 // 8 = 1 3 4
24 // 9 = 2 3 4
25 // ... (if there are more than 5 elements in the set).
26 // Notice that the encoded value does not depend on the number of elements
27 // [a0, a1, ..., a(p-1)] becomes
28 // the integer binomial(a0,1) + binomial(a1,2) + binomial(a2,3) + ...
29 // example: 10 = 0 1 5. This is binomial(0,1) + binomial(1,2) + binomial(5,3) =
30 // 10
31 // example: 9 = 2 3 4. 9 == 2 + binomial(3,2) + binomial(4,3) = 2 + 3 + 4
