116 // 1. call this function with lwork == -1, at exit, work[0] contains optimal
117 // size of 'work'
118 // in number of doubles? or number of bytes? probably number of
119 // doubles...
120 // 2. after that, allocate space for that size, recall the function.
121 // the answer is encoded in the following manner:
122 // Q = H(1) H(2) ... H(k), where k = min(m,n).
123 // H(i) has the form: I - tau * v * v'

125 // tau is a real scalar
