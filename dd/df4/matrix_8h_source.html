311 const Matrix *M); /* drg: connected as rawCoefficients*/
312/* Given:
313 * vars : a list of variable indices in the (common) ring R of monoms and M
314 * monoms : a map R^b --> R^a such that each column has exactly one monomial
315 * which is only in the variables in 'vars'.
316 * M : a map R^c --> R^a such that every (module) monomial of each column of M
317 * matches one of the columns of 'monoms', in the variables 'vars'.
318 *
319 * Returns: a matrix C : R^c --> R^b such that
320 * (i) the entries of C do not involve the variables in 'vars', and
321 * (ii) monoms * C == M
322 *
323 * Assumptions on rings: if R is non-commutative, then the variables in 'vars'
324 * should commute with the variables outside of 'vars'.
325 *
326 * If each column of monoms has more than one monomial, or if variables other
327 * than those in 'vars' occur, then only the first monomial is used, and the
328 * other variables are ignored. If a monomial occurs twice, then one of them
329 * will be used (which one is left undefined)
